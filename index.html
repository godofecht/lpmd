Executable Literate Programming

Literate Python Markdown (LPMD) allows you to write and execute interactive Python programs directly in markdown files. Unlike traditional literate programming that only displays code, LPMD actually runs your markdown documents as executable programs.

Get Started

Key Features

Cell-Based Execution

Code organized in executable cells with dependency management. Each cell can depend on others, ensuring proper execution order.

Smart Dependencies

Cells run in dependency order using topological sort. Define which cells depend on others for complex execution flows.

Variable Persistence

Share variables between cells with explicit persistence. Variables defined in one cell are available in dependent cells.

Markdown Native

Pure markdown format that's readable in any editor. Code executes without affecting the document's appearance.

Web Integration

Generate HTML output and run in browsers. Includes web interface for interactive development.

Audio Processing

Built-in support for audio processing examples, including the 4096-band multi-band compressor.

How LPMD Works

Literate Python Markdown uses special cell markers to define executable code blocks:

LPMD Syntax


```python
print("Hello from LPMD!")
x = 10
y = 20
```


```python
result = x + y
print(f"Sum: {result}")
```


```python
print(f"The final result is: {result}")
```

Execution Flow

1. The setup cell runs first, defining variables x and y

2. The compute cell runs next (depends on setup), calculating the sum

3. The display cell runs last (depends on compute), showing the result

Expected Output

Hello from LPMD!
Sum: 30
The final result is: 30

How LPMD Works

1

Write Markdown

Create a markdown file with special cell markers that define executable code blocks

2

Define Dependencies

Specify which cells depend on others using the 'depends' parameter in cell markers

3

Execute

Run the LPMD executor which parses cells and executes them in dependency order

4

View Results

See the output of your executable document, with variables shared between cells
